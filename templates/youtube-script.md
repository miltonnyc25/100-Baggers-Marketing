# YouTube Script Template

## Structure

```
[0:00-0:30] HOOK
- Open with the most surprising finding
- "Most investors think X, but the data shows Y"
- Visual: Key chart or number on screen

[0:30-2:00] CONTEXT
- Company overview (30s)
- Why this analysis matters now (30s)
- What we'll cover (agenda, 30s)

[2:00-8:00] CORE ANALYSIS (3-4 sections)
- Section 1: [Title] — [Key insight with data]
- Section 2: [Title] — [Key insight with data]
- Section 3: [Title] — [Key insight with data]
- Section 4: [Title] — [Key insight with data]
(Each section: insight → data → implication, 90s each)

[8:00-9:30] RISKS & WHAT COULD GO WRONG
- Bear case summary
- Key risk factors

[9:30-10:00] CONCLUSION & CTA
- One-sentence takeaway
- "Full report link in description"
- Like/subscribe prompt
```

## Guidelines
- Total length: 5-15 minutes
- Mark visual cues: [SHOW: chart], [CUT TO: table], [B-ROLL: product]
- Write for speaking — short sentences, conversational
- Include timestamps for description
- Thumbnail concept: one face + one number + ticker
