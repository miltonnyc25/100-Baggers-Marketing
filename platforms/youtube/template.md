# YouTube Script Template

## Structure
[0:00-0:30] HOOK
- Open with the most surprising finding
- "Most investors think X, but the data shows Y"

[0:30-2:00] CONTEXT
- Company overview + why this analysis matters now

[2:00-8:00] CORE ANALYSIS (3-4 sections)
- Each section: insight -> data -> implication

[8:00-9:30] RISKS & WHAT COULD GO WRONG

[9:30-10:00] CONCLUSION & CTA

## Guidelines
- Total length: 5-15 minutes
- Mark visual cues: [SHOW: chart], [CUT TO: table]
- Write for speaking -- short sentences, conversational
- Include timestamps for description
